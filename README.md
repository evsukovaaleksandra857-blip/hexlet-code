# Brain Games

Brain Games — это набор консольных игр для развития логики и математических навыков.  
Каждая игра предлагает решить 3 задачи подряд. Если игрок ошибается — игра завершается.

## Установка

Требования:
- Python 3.10+
- [uv](https://docs.astral.sh/uv/) — менеджер пакетов

Установка:
```bash
uv tool install git+https://github.com/evsukovaaleksandra857-blip/hexlet-code

```
